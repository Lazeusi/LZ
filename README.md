# Web Development Learning Project (HTML, CSS, JavaScript)

This repository contains simple beginner-level code examples to practice web development using HTML, CSS, and JavaScript.

## 📁 Files Included

- Learn.HTML – HTML practice file
- Learn.CSS – Basic CSS styling
- Learn.JS – Simple JavaScript functionality

## 🚀 How to Run

You can open the project in two ways:

### Option 1: Run in Browser
Just double-click on Learn.HTML to open it in your browser.

### Option 2: Use VS Code with Live Server
If you're using VS Code:
1. Install the "Live Server" extension
2. Right-click Learn.HTML → "Open with Live Server"
